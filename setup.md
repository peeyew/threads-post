# Threads投稿スクリプト セットアップ手順

## 1. Google Sheetsの準備

1. Google Sheetsで新しいスプレッドシートを作成
2. シート名を `posts` に変更
3. 拡張機能 → Apps Script を開く

## 2. スクリプトの貼り付け

1. Apps Scriptエディタに `threads_poster.gs` の内容を全てコピー＆ペースト
2. 保存（Ctrl+S / Cmd+S）

## 3. Script Propertiesにトークンを設定

1. Apps Scriptエディタの左サイドバー「プロジェクトの設定」を開く
2. 「スクリプト プロパティ」セクションで以下を追加：

| プロパティ名 | 値 |
|---|---|
| `THREADS_ACCESS_TOKEN` | 取得済みのアクセストークン |
| `THREADS_USER_ID` | （次のステップで取得） |

## 4. ユーザーIDの取得

1. アクセストークンだけ設定した状態でエディタの関数選択から `getMyUserId` を選んで実行
2. ポップアップにユーザーIDが表示される
3. そのIDを `THREADS_USER_ID` に設定

## 5. シートの初期化

1. 関数選択から `initSheet` を選んで実行
2. ヘッダー行（No. / 本文 / ステータス / 投稿日時）が自動設定される

## 6. 投稿の使い方

| 操作 | 内容 |
|---|---|
| 本文を入力 | B列に投稿テキストを入力 |
| ステータスを変更 | C列を `投稿する` に変更 |
| スクリプト実行 | `postToThreads` を実行 |
| 結果確認 | C列が `投稿済` になり、D列に日時が入る |

### ステータスの種類

| ステータス | 意味 |
|---|---|
| 待機中 | まだ投稿しない |
| 投稿する | 次回実行時に投稿される |
| 投稿済 | 投稿完了（日時記録あり） |
| エラー | 投稿失敗（D列にエラー内容） |

## 7. ボタンを設置する場合（任意）

1. Google Sheetsでシートを開く
2. 挿入 → 図形描画 → 図形を作成して「投稿実行」などのボタンを作成
3. 図形を右クリック → スクリプトを割り当て → `postToThreads` を入力

## 注意事項

- アクセストークンの有効期限に注意（切れた場合は再取得が必要）
- Threads APIには投稿レート制限あり（1ユーザー500件/日）
- `THREADS_ACCESS_TOKEN` はScript Propertiesに保存しコードに直書きしない
